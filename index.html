青春期处理

发表于 2020-11-24 更新于 2020-11-28 分类于 教育方针 Valine：
本文字数： 828 阅读时长 ≈ 1 分钟

- 12、13岁开始 - 父母的点到为止 - 到了青春期，面对孩子，家长要把嘴闭上，关键的时候说上一句。 - 如果对方犯了错误，不要逼着对方承认，跟他说过，他自己下次就不会做了。 - 青春期的孩子不轻信人言，但是亲信书本，影视，来弥补对现实的想象。 - 这个时期要让他交朋友 - 让孩子学点钢琴，搞体育 - 这个时期受欺负，不会找老师去，也有不告诉家长，最好的方式是找同伴，他们会认为结交了坏孩子，才不会被欺负。 - 父母命令的口气变成商量的口气，平等，商量。 - 把孩子当做同事和朋友来对待，说话的 - 让孩子参与家中的大事，就算你能解决，你也要听听孩子的意见。

慈不带兵 义不养财

发表于 2020-11-22 更新于 2020-11-28 分类于 人性 Valine：
本文字数： 2.6k 阅读时长 ≈ 2 分钟

我在运行企业的过程中，有很多的感悟，其中就是这句，慈不掌兵义不掌财。 我百度检索后就看到这篇文章。写的很不错，其实对一个人的不慈是对整个团队的慈。对不随便讲义气的管钱的人，才是能够让财富不断积累的人。这就是局部和整体的关系。做生意，有义气的掺杂，只能过早的分崩离析。 弱肉强食的环境里，竞争是极为残酷的。生存或者毁灭，这是身处其境者需要经常考虑的问题。面对敌人，如果过于仁慈就容易被敌人所趁，害死自己和部下；面对部下，即使他们犯了错，违背你的规矩，你依然一味的过于仁慈，就难以树立威信，无法将他们指挥得形如臂指。管理上的狠是为了出效益，也就是说，对部下的“不慈”事实上就是对团队的“慈”。为了保证生存，为了获取胜利，适当的“不慈”是非常有必要的。

管理应该从企业家传记入门

发表于 2020-11-22 更新于 2020-11-28 分类于 人性 Valine：
本文字数： 454 阅读时长 ≈ 1 分钟
