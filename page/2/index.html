大学老师

发表于 2020-11-18 更新于 2020-11-20 分类于 大学生活
本文字数： 496 阅读时长 ≈ 1 分钟

代课老师一般只带你一学期的课程，不过大学的英语老师还有可能带课四个学期。代课老师一般会在第一节课给出他/她的联系方式，电话或者邮箱。大学老师手中掌握着丰富的资源：各科的科学学习方式、专业浏览网站、该学科的前沿问题、拥有国内本专业其他老师的人脉关系。和老师相处必须牢记的一个原则：尊师重道。尊师表现：上课前擦干净黑板、积极回答老师的问题、认真完成作业。虽然这些都是陈芝麻烂谷子了，但熟知不代表能做到，做到不代表能坚持做下来。师生之间本该互相尊重，老师能多角度的了解你是一个怎样的人，因为他/她在意你。重道表现：敢于和老师交流学术问题，向老师提问，质疑老师。好老师喜欢问问题的学生，所以你不要担心“问题多了老师会烦”诸如此类的问题。

大学学生会

发表于 2020-11-18 更新于 2020-11-20 分类于 大学生活
本文字数： 554 阅读时长 ≈ 1 分钟

学生会有两类：院（系）学生会和校学生会。校学生会是全校同学的学生会，竞争相对会更激烈；院（系）学生会直接和院、系领导打交道，对你今后的大学生活帮助会很大。如果你有当校学生会主席的计划可以先去校学生会锻炼下，一旦胜任对于你是一个很好的磨炼，认识的人会很多，跟各个学校之间交流机会也多，学校内部消息知道的也及时，信息的获取某种意义上讲就是财富。院系的学生会成员接触的是院、系领导和老师，评优、入党会优先，遇到困难也好找领导帮忙。相对而言，校学生会综合性很强，即使校老师想帮你，可能有时也有心而力不足，毕竟校团委老师和你们院系的院长或者党委书记、代课老师之间并不是直接的上下级隶属关系。

有人会跟你说学生会很黑，不要惊讶。学生会里面的确会有很多利益牵扯，班委之间有时也会这样。道理很简单：奖励名额有限，每个人都想拿到，能力相差无几的情况下只能凭谁更讨人喜欢了。所以你能做的是先修炼好自己。古话说的一点也没错：打铁还需自身硬；其次适当的保护自己。害人之心不可无，但防人之心必须有。这里点到为止。

跟这些学生会的部长、主席除了工作上有来往之外，如果能建立起感情不浅的私交就更好了。那怎样建立私交呢？这就要看你的处事风格和方法。学会适当地向异性领导示弱很重要。 总之，一句话，如果能在学生会如鱼得水，你会比别人更快的成长进步~
