大学老师

发表于 2020-11-18 分类于 大学生活
本文字数： 496 阅读时长 ≈ 1 分钟

代课老师一般只带你一学期的课程，不过大学的英语老师还有可能带课四个学期。代课老师一般会在第一节课给出他/她的联系方式，电话或者邮箱。大学老师手中掌握着丰富的资源：各科的科学学习方式、专业浏览网站、该学科的前沿问题、拥有国内本专业其他老师的人脉关系。和老师相处必须牢记的一个原则：尊师重道。尊师表现：上课前擦干净黑板、积极回答老师的问题、认真完成作业。虽然这些都是陈芝麻烂谷子了，但熟知不代表能做到，做到不代表能坚持做下来。师生之间本该互相尊重，老师能多角度的了解你是一个怎样的人，因为他/她在意你。重道表现：敢于和老师交流学术问题，向老师提问，质疑老师。好老师喜欢问问题的学生，所以你不要担心“问题多了老师会烦”诸如此类的问题。

大学兼职

发表于 2020-11-18 分类于 大学生活
本文字数： 2k 阅读时长 ≈ 2 分钟

满18岁之后你就是法律意义上的成人了，但独立生活的能力却不一定具备。大学是一个广阔的平台，在这里你就是主人。好多同学上大学之后，出于孝心和锻炼自己的想法，想通过兼职来减轻家里负担或者赚一些零用钱好攒钱旅行等等。但是，同样是兼职也有高低之分。这里的“高低”并不是指行业的贵贱，体面与否；而是指如果不智慧思考就胡乱找个兼职干，那就是低等的兼职。比如，你是学设计专业的，你却找了一个发传单的工作。再比如，你是学营销的，你却找了个家教的工作，虽然你在代理家教的过程中确实是用自己的知识赚钱，因为营销的本质是点对面，你这是在点对点的销售自己。评价兼职含金量高低的标准之一是：所做工作是否与你的专业相关；标准之二，所做工作是否能够弥补自己的缺点。比如，你性格比较内向，但是想变得稍微外向一些，敢于和陌生人交流，那选择去做销售或者发传单就是一个锻炼自己的机会。
