大学课程

发表于 2020-11-18 更新于 2020-11-20 分类于 大学生活
本文字数： 1.7k 阅读时长 ≈ 2 分钟

大学学习实行学分制度，修满学分方可毕业，言外之意：如果你提前修满学分（所修学分达到学校的要求，最低要达到各个方向要求修满的学分，多则无上限）理论上可申请提前毕业，至于批准与否则取决于学校。大学课程分为选修课与必修课，每门课程都对应着相应学分，学分越高，代表该门课程越重要。学分一般与绩点挂钩，学分绩点反映整个大学期间学习成绩水平，通过公式来进行计算，一般学生手册上会有公式，各个学校的算法不同。绩点在大部分高等学校的研究生保送考核或者申请出国留学时，是一个必要条件。需要注意的一点是：不同课程的不同学分某种意义上代表了学习时你需要投入的精力与注意度。
